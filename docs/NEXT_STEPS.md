# Prochaines etapes

1. Comprendre `frontend/src/App.jsx` et son role actuel dans l'interface.
2. Ajouter un input controle avec `useState`.
3. Faire en sorte que le bouton `Envoyer` ajoute un message utilisateur dans la chat-box.
4. Ajouter CORS cote backend si necessaire pour autoriser l'appel depuis Vite.
5. Connecter React a `POST /chat`.
6. Afficher la reponse de NEXUS dans l'interface.
7. Ameliorer le design NEXUS une fois le flux de chat fonctionnel.
8. Ajouter un mode sombre/clair avec variables CSS et sauvegarde dans `localStorage`.
9. Ajouter un historique local avec `localStorage`.
10. Ajouter SQLite seulement quand le chat marche deja correctement.
